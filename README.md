🧠 AI Mental Health Assist Chatbot

📌 Overview

AI Mental Health Assist is a full-stack intelligent chatbot designed to provide real-time emotional support and mental health guidance. The system leverages advanced Natural Language Processing (NLP) models to understand user input and generate empathetic, context-aware responses.

---

🚀 Features

* 💬 Real-time AI chatbot interaction
* 🧠 Context-aware responses using OpenAI GPT
* 😊 Sentiment and emotion analysis using Hugging Face
* 🔐 Secure authentication with JWT
* 🗂️ Chat history storage and retrieval
* 📱 Responsive UI for mobile and desktop

---

🛠️ Tech Stack

Frontend

* Angular 16+
* HTML5, CSS3, TypeScript
* Bootstrap / Angular Material

Backend

* Node.js
* Express.js
* OpenAI API
* Hugging Face Transformers

Database

* MongoDB with Mongoose

Tools

* VS Code
* Postman
* Git & GitHub

---

⚙️ Architecture

* Frontend communicates with backend via REST APIs
* Backend processes user input and interacts with AI models
* MongoDB stores user data and chat history
* AI models generate intelligent and context-aware responses

---

🔄 Workflow

1. User sends a message via chat UI
2. Request is sent to Node.js backend
3. Backend processes input and calls:

   * OpenAI GPT (for response generation)
   * Hugging Face (for sentiment analysis)
4. Response is returned and displayed to the user
5. Chat history is stored in MongoDB

---

🔐 Security

* JWT-based authentication
* Secure API key handling using `.env`
* Protected user data and sessions

---

📊 Future Enhancements

* 🌍 Multi-language support
* 📈 Mental health analytics dashboard
* ⌚ Wearable device integration
